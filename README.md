# Train Ticket Booking System (C)

## Objective
Console-based train ticket booking system using C programming.

## Features
- Book, cancel, view tickets
- File-based storage
- Multithreading for seat availability
- Modular design with Makefile

## Concepts Used
- Struct, Union, Enum
- Function Pointers
- File Handling
- Pthreads
- Memory Management
- Extern, Typedef
- Loops & Switch Case
